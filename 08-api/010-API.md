## Quick Start

```bash
# Health check
curl http://localhost:3000/api/v1/health

# Com autenticação
curl -H "Authorization: Bearer $TOKEN" \
  http://localhost:3000/api/v1/core/users
```

---

## Base URL

```
Development: http://localhost:3000/api/v1
Production:  https://api.lockdown.app/api/v1
```

## Endpoints

| Grupo | Prefixo | Descrição |
|-------|---------|-----------|
| Core | `/core/*` | Users, Guilds, Members |
| Moderation | `/moderation/*` | Bans, Kicks, Mutes |
| Config | `/config/*` | Guild settings |
| Audit | `/audit/*` | Audit logs |
